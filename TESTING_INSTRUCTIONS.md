## 🔍 Как проверить

### 1. Откройте консоль браузера (F12)

Перед тестированием откройте DevTools → Console

### 2. Перейдите к сотруднику

```
/hr/employees → клик на сотрудника
```

### 3. Проверьте предпросмотр

1. Кликните на иконку "глаз" (Просмотр)
2. В консоли увидите логи:
   ```
   👁️ Preview clicked: {id, type, label}
   📡 Fetching document content...
   📥 Received: {hasContent: true, hasUrl: true}
   ✅ Showing content in modal
   ```

3. Должна открыться модалка с документом

### 4. Проверьте удаление

1. Кликните на иконку "корзина" (Удалить)
2. Для черновика: подтвердите удаление
3. Для подписанного: увидите ошибку "❌ Нельзя удалить подписанный документ"

### 5. Проверьте генерацию

Новые кнопки в секции "Сформировать документ":
- 📘 Трудовой договор
- ✅ Приказ о приеме
- 📝 Заявление
- 🕐 Отпуск

## ⚠️ Если модалка не открывается

Проверьте логи в консоли:
- Если видите `📡 Fetching...` но нет `📥 Received` - проблема с API
- Если видите `❌ Preview error` - проверьте Network tab
- Если модалка пустая - проблема с `content`

## 🔧 Debug

Откройте Network tab и проверьте:
```
GET /documents/:id/content
Status: 200 OK
Response: { scan_url: "...", content: "<html>...", type: "html" }
```

Если 404 или 500 - проблема на backend
