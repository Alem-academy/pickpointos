# 🗺️ Карта навигации HR кабинета

## ✅ Все роуты HR

| Роут | Страница | Описание | Связи |
|------|----------|----------|-------|
| `/hr` | Dashboard | Дашборд HR | → Applications, → Employees |
| `/hr/applications` | Applications | Воронка найма (Канбан) | → New Employee |
| `/hr/employees` | Employees | Список сотрудников | → New Employee, → Profile |
| `/hr/new-employee` | New Employee | Анкета найма | → Profile (после создания) |
| `/hr/employees/:id` | Employee Profile | Профиль сотрудника | ← Employees |
| `/hr/timesheet` | Timesheet | Табель | - |
| `/hr/discipline` | Discipline | Дисциплина | - |
| `/hr/templates` | Templates | Шаблоны документов | - |

## 🔄 Основные workflow

### 1. Найм сотрудника (Onboarding)

```
/hr (Dashboard)
    ↓
/hr/applications (Воронка найма)
    ↓ "Создать заявку"
/hr/new-employee (Анкета: 3 шага)
    Step 1: Личные данные
    Step 2: Условия работы  
    Step 3: Документы
    ↓ "Создать"
/hr/employees/:id (Профиль)
    → Генерация договора
    → Подписание
```

### 2. Просмотр сотрудников

```
/hr (Dashboard)
    ↓ "Команда"
/hr/employees (Список)
    ↓ Клик на сотрудника
/hr/employees/:id (Профиль)
    → Документы
    → История
    → Дисциплина
```

### 3. Быстрый доступ (Sidebar)

```
Sidebar (постоянно виден):
├── 📊 Дашборд (/hr)
├── 📄 Заявки (/hr/applications)
├── 👥 Сотрудники (/hr/employees)
├── ➕ Новый сотрудник (/hr/new-employee)
├── 📋 Шаблоны (/hr/templates)
└── 🏢 Новый ПВЗ (/operations/new-pvz)
```

## 🔗 Связность проверена ✅

Все ссылки работают корректно:
- ✅ Dashboard → Applications
- ✅ Dashboard → Employees
- ✅ Applications → New Employee
- ✅ Employees → New Employee
- ✅ Employees → Profile
- ✅ Profile → Employees (back button)
- ✅ New Employee → Profile (после создания)
- ✅ Sidebar → все основные разделы

## 📱 UX рекомендации для HR

### Для быстрого найма:
1. **Sidebar** → "Новый сотрудник" (прямой доступ)
2. Или **Dashboard** → "Создать заявку"
3. Или **Applications** → "Создать заявку на прием"

### Для работы с документами:
1. **Employees** → выбрать сотрудника
2. **Profile** → вкладка "Документы"
3. Загрузить/Сгенерировать → Подписать

### Для массовых операций:
1. **Employees** → поиск/фильтры
2. Клик на карточку → профиль
3. Быстрые действия из профиля

## 🎯 Что работает хорошо в Employee Profile

1. **Четкая структура** - вкладки (Документы, История, Дисциплина)
2. **Визуальные карточки** - документы с иконками
3. **Быстрые действия** - кнопки генерации/загрузки
4. **Статусы** - цветовое кодирование
5. **Модальные окна** - загрузка с выбором типа

## 💡 Идеи для улучшения New Employee

Можно применить лучшие практики из Profile:

1. **Предзаполнение** - если кандидат из Applications
2. **Прогресс бар** - показывать шаг 1/2/3
3. **Сохранение черновика** - localStorage
4. **Валидация в реальном времени** - зеленые галочки
5. **Подсказки** - tooltips для каждого поля
6. **Быстрая загрузка** - drag & drop для документов
7. **Чек-лист** - какие документы обязательны
