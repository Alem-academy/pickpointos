### Сотрудники

- 5 сотрудников → ИП Жасмин
- 2 сотрудника → ИП Ориентал
- 1 сотрудник → без ПВЗ (требуется назначить)

## 🔄 Типы перемещений

### Временная замена
- **Поле:** `current_pvz_id`
- **Юрлицо:** Не меняется
- **Договор:** Не требуется

### Постоянный перевод
- **Поле:** `main_pvz_id`
- **Юрлицо:** Меняется автоматически
- **Договор:** Расторжение + новый

См. `TRANSFER_GUIDE.md` для деталей.

## ⚠️ Что нужно сделать

### 1. Обновить ИБАНы на актуальные

```sql
-- Проверить текущие ИБАНы
SELECT name_short, iban, bank_name FROM employers;

-- Обновить на актуальные (из банковских договоров)
UPDATE employers SET iban = 'KZ____________________' WHERE name_short = 'Жасмин';
UPDATE employers SET iban = 'KZ____________________' WHERE name_short = 'Ориентал';
UPDATE employers SET iban = 'KZ____________________' WHERE name_short = 'PVZ.kz';
```

### 2. Проверить генерацию документов

1. Зайти в HR панель
2. Выбрать сотрудника
3. Сгенерировать трудовой договор
4. Проверить шапку документа с реквизитами юрлица

## 🔄 Как работает

1. При приеме сотрудник привязывается к ПВЗ
2. Система автоматически определяет юрлицо ПВЗ
3. Документы генерируются с реквизитами этого юрлица
4. При переводе на другой ПВЗ юрлицо обновляется автоматически
